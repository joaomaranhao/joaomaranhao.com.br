# How to install Z Shell (zsh) on linux

Z shell is a powerful shell with a lot of plugins that make everything easier. From auto-suggestions to an ssh-agent loader, it makes our workflow more efficient.

## Installation

Most distros already have zsh in their repository, which makes the installation process pretty easy.

### Debian (Ubuntu, Linux Mint, PopOS, etc)

```
sudo apt install zsh
```
